# Advent of Code 2025 in Node.js

This repo is my personal attempt at solving the [Advent of Code](http://adventofcode.com/) 2025 set of problems using Node.js.

## Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine (version 14.x or higher recommended)

## How to run a script

1. Clone this repository:

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

2. Make sure you have Node.js installed.

3. Run a specific JavaScript file with Node.js:

```bash
node path/to/your-script.js
```

4. Follow the instructions in the terminal to input data if needed.
